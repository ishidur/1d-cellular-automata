# 1d-cellular-automata
python 1d cellular automata visualization tool
![sample](./rule135.png)

# How to use
## Run
```
python 1d-cellular-automata.py
```

## Change rule num
go to [here](https://github.com/ishidur/1d-cellular-automata/blob/master/1d-cellular-automata.py#L9)
and change "ruleNo"
